# Lispy

An implementation of a Lisp from [Build your own Lisp](http://www.buildyourownlisp.com) written in Ada 2012.

# Building

## GNAT

```bash
$ git clone git@github.com:Lucretia/lispy.git
$ cd lispy/build/gnat
$ make
```

# Dependencies

Ada 2012 compiler.

[MPC](https://github.com/orangeduck/mpc.git)

## Tested with

FSF GNAT 7.3.0
